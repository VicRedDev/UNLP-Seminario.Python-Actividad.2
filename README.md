# UNLP - Seminario de Python 2026 - Actividad 2

Este repositorio es la entrega del ejercicio 10 de la practica/actividad 2 del seminario de Python 2026 de la Facultad de Informatica de la UNLP

## Ejecucion del repositorio

### Requisitos

- Python 3.13.* 
- Git
- python3-venv(requisito en Linux).

### Pasos de ejecucion:

**Clonar repositorio**

```
git clone https://github.com/VicRedDev/UNLP-Seminario.Python-Actividad.2.git
cd UNLP-Seminario.Python-Actividad.2
```

**Crear entorno**
```
python3 -m venv .venv
```

**Activar entorno**

- En Linux(bash):

    ```
    source .venv/bin/activate
    ```
- En Windows(CMD)

    ```cmd
    .venv/Scripts/activate
    ```

**Instalar requerimientos**
```
python3 -m pip install -r requirements.txt
```

**Abrir notebook**
```
jupyter-notebook notebooks/exercises.ipynb
```

Al hacer esto ultimo, se abrira el notebook, y al seleccionar el boton para ejecutar todas las celdas, se podran ver los resultados del codigo en la ultima celda.
